# 📱 About The Project

FoodyMind Admin App is an Android application developed using **Kotlin** and **MVVM Architecture**. It allows restaurant administrators to manage food items, customer orders, delivery status, and restaurant operations efficiently using **Firebase**.

The application is designed with a clean architecture and modern Android development practices, making it suitable for academic and professional portfolios.

---

# ✨ Features

- 🔐 Admin Login
- 🍔 Add Food Items
- 📝 Update Food Items
- ❌ Delete Food Items
- 📋 View All Items
- 🛒 Manage Pending Orders
- 🚚 Out For Delivery
- ✅ Completed Orders
- 👥 Create New Admin/User
- 💰 View Total Earnings
- 📍 Manage Delivery Status
- 🔥 Firebase Integration

# 🏗 Architecture

The project follows the **MVVM (Model-View-ViewModel)** architecture for better code organization, maintainability, and scalability.

```
UI (Activity / Fragment)
        │
        ▼
    ViewModel
        │
        ▼
    Repository
        │
        ▼
Firebase Realtime Database
Firebase Storage
Firebase Authentication
```
